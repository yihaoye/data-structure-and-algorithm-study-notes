## 白板绘图、数据结构手绘
如何在白板（文本文件）上绘制常见的数据结构并且解释思路  

哈希表
```
key1 -> val1
key2 -> val2
```

集合
```
{2, 3, 1, 5, 0, 4}
```

顺序表、数组
```
[1, 2, 3, 4, 5]
```

链表
```
1 -> 2 -> 3
```

环形链表
```
1 -> 2 -> 3
      |   |
       <--   
```

二叉树
```
     4
    /  \
   12   5
         \
         10
```

双指针
```
  [1, 2, 3, 4, 5]
   ^           ^
 left        right
```

栈
```
10
5
2
bottom
```

无向图
```
1 - 4
|   |
2 - 3 - 10
```

有向图、状态机
```
1 -> 4
|    |
v    v （字母 v）
2 <- 3 -> 10
```

前缀树
```
     a
    p c
   p   t (end)
  l
 e (end)
```
