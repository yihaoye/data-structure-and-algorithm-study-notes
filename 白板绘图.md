## 白板绘图
参考：https://interview-science.org/%E7%AE%97%E6%B3%95%E6%8C%87%E5%8D%97/%E6%96%87%E6%A1%A3%E7%BB%98%E5%9B%BE  
在白板（文本文件）上绘制常见的数据结构并且解释思路  

链表
```
1 -> 2 -> 3
```

环形链表
```
1 -> 2 -> 3
      |   |
       <--   
```

二叉树
```
     4
    /  \
   12   5
         \
         10
```

双指针
```
  [1, 2, 3, 4, 5]
   ^           ^
 left        right
```

栈
```
10
5
2
bottom
```

无向图
```
1 - 4
|   |
2 - 3 - 10
```

有向图
```
1 -> 4
|    |
v    v （字母 v）
2 <- 3 -> 10
```

前缀树
```
     a
    p c
   p   t (end)
  l
 e (end)
```
